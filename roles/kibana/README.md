# Task

**Debian only**

Install Kibane the Elastic stack front end.

 * Install Kibana
 * `/etc/kibana/kibana.yml` - Kibana configuration file.
 * Add and start the daemon.

# Dependencies

  * OpenJDK 8 JRE
  * Elastic repositories

# Variables

 * `elastic_backend_ip` - IP address of the elastic stack back end server
   * Default: 127.0.0.1
